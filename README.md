# 🎯 Interview Coach Pro

A full-stack Django-based interview preparation platform designed to help students and aspiring software engineers prepare for technical and behavioral interviews through MCQs, coding challenges, written-answer evaluations, personalized feedback, and performance analytics.

---

## 🚀 Overview

Interview Coach Pro simulates a real interview preparation environment by combining multiple assessment formats into a single platform. Users can practice technical concepts, improve problem-solving skills, receive structured feedback, and track their progress over time.

The platform focuses on core Computer Science and Software Engineering interview preparation, making it suitable for B.Tech CSE students, fresh graduates, and aspiring developers.

## 🚀 Installation

Clone the repository:

```bash
git clone https://github.com/anirudhsinghsalwa/AI-Powered-Interview-Coach.git
```

Navigate to project directory:

```bash
cd AI-Powered-Interview-Coach
```

Create virtual environment:

```bash
python -m venv venv
```

Activate virtual environment:

```bash
venv\Scripts\activate
```

Install dependencies:

```bash
pip install -r requirements.txt
```

Run migrations:

```bash
python manage.py migrate
```

Start server:

```bash
python manage.py runserver
```
